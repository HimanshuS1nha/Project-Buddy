# Project Buddy

Manage your projects seemlessly with full control.

## Get started

1. Install dependencies

   ```bash
   cd client
   npm install
   cd ..
   cd server
   npm install
   ```

2. Start the app

   ```bash
    cd client
    npm run dev

    cd ..

    cd server
    npm start
   ```

## Features

- Login/Logout
- Create Projects
- Create tasks
- Drag and Drop tasks
- Realtime chat
- Add team members

## TECH STACK USED

- React
- ExpressJS
- NodeJS
- MongoDB
